零基础接触这玩意可真痛苦，怎么还得学html css js？？？然后怒开B站，恼了三天三夜。

本小程序初衷:造福大家（钱包）
设计灵感来源于"墨墨背单词"，作者高中一直在用，但是奈何不能带手机，当时也没有时间来开发这个

由于单词表内涵大量单词汉译和短语，所需空间大小为6MB，请酌情考虑清理空间。


开发文档即使用说明: https://iot.mi.com/vela/quickapp/zh/guide/ 请使用AIoT IDE来编译此小程序

words.js内格式均为{ "english": "terrible", "phonetic": "[ˈterəbl]", "chinese": "a. 可怕的 | 糟糕的，差劲的 | feel terrible 感觉很难受 | terrible mistake 严重的错误" }三大部分

1. 内置高中6200词汇表(默认)，有需求可以联系我：LonelyBing@outlook.com
2. 每个单词四个“灵魂选项”：
   - 熟知（绿色）——这个词已经是我兄弟了
   - 认识（白色）——见过面，不算熟
   - 模糊（黄色）——似曾相识，但不确定
   - 忘记（红色）——对不起，我们第一次见

3. 熟知单词今后练习中不会再出现。
4. 除熟知外，其余统统归为“待练习单词”，每天从这里面 随机抽50个，

5. 认识类：今天练的时候会重复3次

6. 模糊类：今天练的时候会重复5次

7. 忘记类：今天练的时候会重复7次

8. 还有复习功能，能看近7日学习统计，重在坚持。
